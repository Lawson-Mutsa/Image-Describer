# Overview

To further my learning, I intend to fully understand both front-end and back-end development to become an AI Software Developer.

The software I developed is an Image Describer web application that uses Artificial Intelligence to analyze an uploaded image and generates a description of its contents. The application allows users to upload an image and receive an AI-generated description.

The purpose of this project was to gain hands-on experience integrating Artificial Intelligence into web applications.
