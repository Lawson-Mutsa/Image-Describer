# Image Describer Overview

A web application that describes the content of an image using a hugging face model


[Software Demo Video](https://youtu.be/5Q-vQAoUpVU)

# Development Environment

Languages used are:

* HTML
* CSS
* JavaScript


# Step by Step Guide
* Create an account at [Hugging Face](https://huggingface.co/)
* Create a token after succesful account creation [Create token](https://huggingface.co/settings/tokens). Token type is read
* Copy the token on modal displayed after clicking create token
* Paste the token on line 4 (variable HF_TOKEN value) in the functionality.js file
* Open [Image Describer](https://glittery-starlight-b789c8.netlify.app/)
* Upload an Image
* Image starts to be processed (image description is being generated)
* A description is generated after loading
* The description is displayed at the bottom of the page, after the image
